# Dockerfile

### ENV

```
ENV <nombre variable> <valor de la variable>
ENV <nombre variable>=<valor de la variable>
ENV <nombre variable1> <valor de la variable1> <nombre variable2> <valor de la variable2>
ENV <nombre variable1>=<valor de la variable1> <nombre variable2>=<valor de la variable2>
```

### WORKDIR

```
WORKDIR <ruta del directorio>
```
